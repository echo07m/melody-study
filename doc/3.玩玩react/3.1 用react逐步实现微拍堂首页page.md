### 样式的玩法
1  css 类名 => className 
```js
    function A(){
        return (
            <div className='com-a'>
                组件A
            </div>
        )
    }
```
2 css in js=> style
```js
    function A(){
        return (
            <div style={{
                color:'#000'
            }}>
                组件A
            </div>
        )
    }
```
